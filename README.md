  # Calculadora em Python (Terminal)

Este projeto é uma calculadora simples para terminal, desenvolvida em Python.  
Permite realizar operações básicas entre dois números com validação de entradas e operadores.

---

## Funcionalidades

- Soma (`+`)
- Subtração (`-`)
- Multiplicação (`*`)
- Divisão (`/`)
- Validação se os números digitados são válidos
- Validação do operador digitado
- Evita erros de entrada inválida
- Loop contínuo até o usuário decidir sair

---

## Como usar

1. Abra o terminal ou prompt de comando.  
2. Navegue até a pasta do projeto.  
3. Execute o arquivo da calculadora:

```bash
python calculadora_v1.py
